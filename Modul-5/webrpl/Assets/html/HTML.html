HTML

      HTML (Hyper Text Markup Language) adalah sebuah bahasa pemrograman yang digunakan untuk membuat halaman website. HTML menjadi dasar dari sebuah halaman website. sebuah file HTML di di simpan dengan ekstensi .html dan dapat di eksekusi atau diakses menggunakan web browser (Google Chrome, Mozilla Firefox, Opera, dan lain-lain). Untuk membuat sebuah website tidak cukup hanya menggunakan HTML, diperlukan bantuan CSS, JavaScript dan PHP untuk membuat website yang dinamis. Halaman website yang dibuat hanya menggunakan HTML maka halaman website tersebut di sebut halaman statis karena tidak memiliki aksi atau fungsi- fungsi yang dapat mengelola website.

      Terdapat beberapa hal yang dapat dilakukan dengan HTML sebagai berikut :
      1. Membungkus element-element tertentu sesuai kebutuhan.
      2. Membuat heading atau format judul.
      3. Membuat list.
      4. Membuat tabel.
      5. Membuat paragraf.
      6. Membuat form.
      7. Membuat tombol.
      8. Membuat huruf tebal.
      9. Membuat huruf miring.
      10. Menampilkan Foto.
      11. menampilkan video.
      12. memutar sound.
