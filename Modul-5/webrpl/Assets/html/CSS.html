CSS

  1. PENGENALAN CSS

    Cascading Style Sheets atau disingkat CSS memiliki sifat ”style sheet language” yang berarti bahasa pemrograman yang di gunakan untuk web design. CSS adalah bahasa pemrograman yang di gunakan untuk men-design sebuah halaman website. Dalam mendesign halaman website, CSS menggunakan penanda yang kita kenal dengan id dan class. Seperti yang sudah dijelaskan pada tutorial edisi belajar HTML dasar sebelumnya. Seiring berkembang nya dunia pemrograman dan teknologi, CSS tidak cuma di gunakan di HTML dan XHTML saja. Tapi bisa di gunakan untuk mendesign tampilan aplikasi android. CSS di akses menggunakan id atau class.


  2. FUNGSI CSS

    CSS dapat mengubah font, ukuran font, warna dan format font. Mengatur ukuran layout, lebar, tinggi dan warna element, mengubah tampilan form, membuat halaman website yang responsive dan masih banyak lagi yang dapat di lakukan oleh CSS.
    Untuk mendesain font dapat dilakukan dengan mendefinisikan font, untuk mengatur warna bisa menggunakan color, margins digunakan untuk mengatur jarak pada luar element tertentu. Mengatur warna atau gambar pada latar belakang bisa menggunakan “background”. Mengatur ukuran font menggunakan “font size”. Jenis font menggunakan “font-family” dan banyak lagi lainnya


  3. PENGGUNAAN CSS

    1. Penamaan dan Menghubungkan dengan CSS

      File css di simpan dengan ekstensi “.css” kemudian di hubungkan kedalam file HTML atau PHP yang ingin kita design dengan CSS menggunakan syntax:

      <

      link rel="stylesheet" type="text/css" href="style.css" />

      Tag di atas di gunakan untuk menghubungkan file HTML dengan file CSS. Syntax di atas terletak pada bagian head file html, yang terdiri dari atribut rel dan type yang di gunakan untuk mendefinisikan bahwa yang di panggil atau yang di hubungkan adalah file Style sheet atau CSS. Kemudian atribut href di gunakan untuk membaca dimana letak file CSS. Pada contoh di atas file style.css terletak satu folder atau direktori dengan file html. Namun jika file css terletak dalam sebuah folder dengan nama foldernya adalah “assets” maka untuk menghubungkannya menggunakan syntax:

      <

      link rel="stylesheet" type="text/css" href="assets/style.css" />
